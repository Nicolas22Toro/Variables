# Taller_2 Variables
 
# Descripcion

Este repositorio contiene el desarrollo de un programa en Java, cuyo propósito es practicar la declaración de variables utilizando distintos tipos de datos y resolviendo dos formulas ya antes declaradas

# Contenido

-Crear y/o cambiar el repositorio en GitHub
-Desarrollar los 2 programas usando las variables de Java
-Gestion de cambio mediante los commits

# Variables

Las variables que debemos incluir con los tipos de datos solicitados son Float, Chair, Loong y tratar de hacer un bucle con el comando While; tambien añadiremos los primitivos tales como int, double y boolean.

# Repositorio 
Con los siguientes comandos podremos trabajar en la creacion de un repositorio en "GitHub" usando la herramienta "Gitbash"
Git clone "URL"
Readme.md
git add readme.md
git commit -m "creacion readme"
git branch -m main
git push -u origin main
Ahora para actualizar nuestro repositorio debemos cambiar los usuarios añadidos con los comandos
git config --global user.mail
git config --global user.name
git config --list para verificar nuestro usuarios y emails añadidos y eliminar los que no estamos usando.

# Programa 

En Visual Studio Code, con ayuda de la extensión del paquete de java creamos un codigo con el cual vamos a declarar 20 variables y les damos a dar uso a traves de un mensaje de texto; el segundo programa solucionara un problema matematico antes ya preescrito utilizando el sistema scanner para poderlo ejecutar y darle un bucle infinito en el cual tenemos las opciones de hacer que el sistema funcione o salir del programa.
